# Brand assets

Caustic visual identity. The mark is an optical caustic: a fan of light rays converging to a cusp.

## Files
- `caustic-mark.svg` - primary mark (near-black, transparent)
- `caustic-mark-dark.svg` - mark for dark backgrounds (off-white)
- `caustic-favicon.svg` - square crop of the mark
- `caustic-favicon-glyph.svg` / `-dark.svg` - simplified glyph for small sizes (<= 32px)
- `favicon-16/32/48.png` - browser favicons
- `icon-180.png` / `icon-512.png` - app / touch / maskable icons
- `og-1200x630.png` - social / Open Graph card
- `avatar-512.png` - organization avatar

## Palette
- Near-black `#0B0B0C`
- Off-white `#ECECE6`
- Electric cyan `#24D6E8` (accent - the caustic curve only)

The full mark's fine rays disappear below ~32px; use the glyph for tiny sizes.
